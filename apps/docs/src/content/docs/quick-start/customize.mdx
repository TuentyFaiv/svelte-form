The built-in components comes with a default look and feel, you can customize it using `setConfig` or the `styles` property in each built-in component and the [`faivform`](/reference/#faivformconfig) function. Also Faivform library comes with a light and dark theme that you can customize too.

## Global

You must use the `setConfig` in the root component of you application. You can check the [`Config`](/reference/#global-config) to see all the options.

### Classes

In the `fields` option you can customize the class names of the built-in components. You can check the [`ContextStyles`](/reference/#contextstyles).

```svelte
<script lang="ts">
  import { setConfig } from "@tuentyfaiv/svelte-form";

  setConfig({
    fields: {},
    // ... themes
  });
</script>
```

### Theme light

To customize the light theme you must use the `light` and `onLight` options. You can check the [`FaivFormTheme`](/reference/#faivformtheme).

```svelte
<script lang="ts">
  import { setConfig } from "@tuentyfaiv/svelte-form";

  setConfig({
    // ... classes
    light: {},
    onLight: {},
    // ... theme dark
  });
</script>
```

### Theme dark

To customize the dark theme you must use the `dark` and `onDark` options. You can check the [`FaivFormTheme`](/reference/#faivformtheme). To detect the dark mode you must use `.dark` class or `data-theme="dark"` in the `html` element.

```svelte
<script lang="ts">
  import { setConfig } from "@tuentyfaiv/svelte-form";

  setConfig({
    // ... classes
    // ... theme light
    dark: {},
    onDark: {},
  });
</script>
```

## Per form

Here is an example of how to customize the look and feel of all the fields per form. You must use the [`styles`](/reference/#faivformconfig) option in the `faivform` function.

```svelte
<script lang="ts">
  import { faivform } from "@tuentyfaiv/svelte-form";

  const form = faivform({
    // ... schema or adapter and context name
    styles: {},
  });
</script>

```
## Per field

Also you can customize the look and feel of a specific field by using the `styles` property in each built-in component. Depending on the component you must use the styles options according to the component. You can check the [`styles definition`](/reference/#styles) of each component.

For example, if you want to customize the look and feel of the `Field` and `Select` components you can do it like this:

```svelte
<script lang="ts">
  import { Field, Select } from "@tuentyfaiv/svelte-form";

  import type { FieldStyles, SelectStyles } from "@tuentyfaiv/svelte-form";

  const fieldStyles: FieldStyles = {
    // ... class names
  };

  const selectStyles: SelectStyles = {
    // ... class names
  };
</script>

<Field styles={fieldStyles} name="some-field" />
<Select styles={selectStyles} name="other-field" />
```
