Faivform is a form library for Svelte. It is built on top of [Svelte](https://svelte.dev/) and [Typescript](https://www.typescriptlang.org/). It is inspired by [Formik](https://formik.org/) and [React Hook Form](https://react-hook-form.com/).

## Installation

To install the latest version of the library, run the following command:
<Tabs>
  <TabItem label="pnpm">
  ```bash frame="none"
  pnpm add @tuentyfaiv/svelte-form
  ```
  </TabItem>
  <TabItem label="npm">
  ```bash frame="none"
  npm install @tuentyfaiv/svelte-form
  ```
  </TabItem>
</Tabs>

## Compatibility

| Tool       | Version   |
|------------|-----------|
| Svelte     | >= 4.0.0  |
| SvelteKit  | >= 1.0.0  |
| TypeScript | >= 5.0.0  |
| Node       | >= 18.0.0 |
| npm        | >= 8.0.0  |
| pnpm       | >= 8.0.0  |
