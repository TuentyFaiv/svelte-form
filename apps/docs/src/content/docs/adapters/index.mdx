Adapters are used to convert the schema to a format that `faivform` can understand. For example, if you want to use Yup, you need to install the corresponding adapter.

You can use the following adapters:

- [Zod](https://zod.dev/)
- [Yup](https://www.npmjs.com/package/yup)

## Installation

First you need to install the adapter you want to use.

<Tabs>
  <TabItem label="pnpm">
    To use Zod:
    ```bash frame="none" 
    pnpm add @faivform/zod
    ```
    To use Yup:
    ```bash frame="none"
    pnpm add @faivform/yup
    ```
  </TabItem>
  <TabItem label="npm">
    To use Zod:
    ```bash frame="none" 
    npm install @faivform/zod
    ```
    To use Yup:
    ```bash frame="none"
    npm install @faivform/yup
    ```
  </TabItem>
</Tabs>


## How to use

To use the adapter you need to import it and pass it to the `faivform` function.

<Tabs>
  <TabItem label="zod">
    ```svelte
    <script>
      import { faivform, Field } from "@tuentyfaiv/svelte-form";
      import { adapter } from "@faivform/zod";
      import { z } from "zod";

      const schema = z.object({
        name: z.string(),
        age: z.number(),
        email: z.string().email(),
      });

      const form = faivform({ fields: adapter(schema) });

      // ... rest of the code
    </script>
    ```
  </TabItem>
  <TabItem label="yup">
    ```svelte
    <script>
      import { faivform, Field } from "@tuentyfaiv/svelte-form";
      import { adapter } from "@faivform/yup";
      import * as yup from "yup";

      const schema = yup.object().shape({
        name: yup.string().required(),
        age: yup.number().required().positive().integer(),
        email: yup.string().email(),
      });

      const form = faivform({ fields: adapter(schema) });

      // ... rest of the code
    </script>
    ```
  </TabItem>
</Tabs>
