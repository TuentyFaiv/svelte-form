# Svelte Forms


### Steps to install
- `npm install @tuentyfaiv/svelte-form`

### Store 
- `import { formStore } from "@tuentyfaiv/svelte-form";`

### Components
- `import { Input, Select, FileInput } from "@tuentyfaiv/svelte-form";`

### Types
- `import type { Fields, Data, SelectOption  } from "@tuentyfaiv/svelte-form";`
